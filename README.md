# How are graceful degradation and service workers related?
Graceful degradation and service workers are related because both of them help keep web applications usable and functional when certain features fail or are unavailable. Graceful degradation is the practice of designing a website so that it's core features still work even when more advanced features are unsupported. Service workers are tools used to enable offline functionality, so they implement graceful degredation.
